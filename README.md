# 行為科學×習慣養成×AI 教學網站

本專案示範了一個結合行為科學、習慣養成理論與人工智慧應用教學的網站。網站分為前台與簡單後台，可用於展示相關文章、追蹤習慣並提供基本的建議機能。後端使用 Python 的 Flask 框架實作，資料庫採用 SQLite，前端使用 Bootstrap 建立簡單的響應式頁面。

## 功能特色

- **前台**
  - 首頁：顯示行為科學、AI 應用與習慣養成相關的文章內容。
  - 習慣追蹤：使用者可以查看預設的習慣、輸入新的習慣並標記完成。

- **後台**
  - 登入簡易管理介面（密碼預設為 `adminpass`）。
  - 新增文章：輸入文章標題與內容後可存入資料庫，在首頁顯示。
  - 新增習慣：輸入習慣名稱與描述，便會出現在習慣追蹤頁面。

## 安裝與執行

1. 建議使用 Python 3.9+ 環境。
2. 安裝依賴套件：
   ```bash
   pip install -r requirements.txt
   ```
3. 初始化資料庫：首次執行時 Flask 會自動產生 `behavior.db`。
4. 啟動儲會器：
   ```bash
   python app.py
   ```
5. 開啟瀏覽器前往 `http://127.0.0.1:5000` 觀看前台。
6. 後台登入：至 `http://127.0.0.1:5000/admin`，輸入預設密碼 `adminpass` 後即可進入管理界面。
